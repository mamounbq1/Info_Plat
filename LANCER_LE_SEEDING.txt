   ┌─────────────────────────────────────────────────────────────────────┐
   │ ÉTAPE 1: OUVRIR LES RÈGLES FIREBASE (⏱️ 2 minutes)                  │
   └─────────────────────────────────────────────────────────────────────┘
   
   1. Aller sur: https://console.firebase.google.com
   2. Sélectionner: eduinfor-fff3d
   3. Menu: Build → Firestore Database → Rules
   4. Remplacer par les règles temporaires (voir ci-dessous)
   5. Cliquer: "Publish"

   RÈGLES TEMPORAIRES À COPIER:
   ┌─────────────────────────────────────────────────────────────────────┐
   │ rules_version = '2';                                                │
   │ service cloud.firestore {                                           │
   │   match /databases/{database}/documents {                           │
   │     match /{document=**} {                                          │
   │       allow read, write: if true;                                   │
   │     }                                                                │
   │   }                                                                  │
   │ }                                                                    │
   └─────────────────────────────────────────────────────────────────────┘

   ┌─────────────────────────────────────────────────────────────────────┐

   └─────────────────────────────────────────────────────────────────────┘
   
   Dans votre terminal:
   
   npm run seed:comprehensive

   ⏳ Patientez... Le script va créer 89 documents dans Firestore.

🆘 PROBLÈMES?

   Erreur: "Permission denied"
      → Les règles Firebase ne sont pas ouvertes. Voir Étape 1.
   
   Erreur: "Firebase not initialized"
      → Vérifier src/config/firebase.js
   
   Les utilisateurs ne peuvent pas se connecter
      → Créer les utilisateurs dans Firebase Authentication
      → Build → Authentication → Add user
   
   Autre problème?
      → Consultez SEEDING_GUIDE.md section "Dépannage"
