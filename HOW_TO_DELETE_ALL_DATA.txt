METHOD 1: FIREBASE CONSOLE (✅ RECOMMENDED - EASIEST!)
─────────────────────────────────────────────────────

This is the simplest method and always works!

Step 1: Open Firestore Console
   🔗 https://console.firebase.google.com/project/eduinfor-fff3d/firestore/data

Step 2: Delete Each Collection

   For "users" collection:
   ├─ Click on "users" in sidebar
   ├─ Click three dots (⋮) menu
   ├─ Select "Delete collection"
   ├─ Type collection name to confirm
   └─ Click "Delete"

   Repeat for these collections:
   ✅ users (24 docs)
   ✅ courses (19 docs)
   ✅ quizzes (11 docs)
   ✅ exercises (10 docs)
   ✅ branches (4 docs)
   ✅ subjects (11 docs)

Step 3: Delete Authentication Users (Optional)
   🔗 https://console.firebase.google.com/project/eduinfor-fff3d/authentication/users
   ├─ Check box at top (select all)
   ├─ Click delete button
   └─ Confirm deletion

Step 4: Verify Deletion
   Run: node check-database-contents.js
   Should say: "✨ Database is empty!"

═══════════════════════════════════════════════════════════════════════════

METHOD 2: USING SCRIPTS (Requires Firebase CLI)
────────────────────────────────────────────────

Prerequisites:
   npm install -g firebase-tools
   firebase login

Step 1: Deploy Open Rules
   firebase deploy --only firestore:rules --project eduinfor-fff3d

Step 2: Wait 60 seconds
   (Rules need time to propagate)

Step 3: Run Deletion Script
   node delete-all-simple.js

Step 4: Verify
   node check-database-contents.js

═══════════════════════════════════════════════════════════════════════════

METHOD 3: FIREBASE ADMIN SDK (Advanced)
────────────────────────────────────────

Step 1: Download Service Account Key
   🔗 https://console.firebase.google.com/project/eduinfor-fff3d/settings/serviceaccounts/adminsdk
   ├─ Click "Generate new private key"
   └─ Save as "serviceAccountKey.json" in project root

Step 2: Run Admin Script
   node delete-all-data-admin.js

Step 3: Follow Prompts
   ├─ Type "DELETE ALL DATA" to confirm
   └─ Choose whether to delete auth users

Step 4: Verify
   node check-database-contents.js
